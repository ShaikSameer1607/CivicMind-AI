# Screenshot Placeholders

This directory should contain the following screenshots to be referenced by the main README.md. Please capture and save the following images with the exact filenames listed below:

- `banner.png`: A hero banner or logo image for CivicMind AI.
- `architecture.png`: A diagram explaining the system architecture and agent pipeline.
- `citizen-dashboard.png`: Screenshot of the main dashboard seen by citizens after logging in.
- `report-page.png`: Screenshot of the report issue form and media upload component.
- `admin-dashboard.png`: Screenshot of the government operations issue management view.
- `ai-analysis.png`: Screenshot showing the AI Copilot panel or agent metrics.
- `analytics.png`: Screenshot of the city intelligence analytics and insights.
- `map-view.png`: Screenshot of the interactive map with hotspots.
